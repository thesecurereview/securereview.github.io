SecureReview

A defense mechanism to mitigate attacks against web-based code review systems such as GitHub and Gerrit.

Main features provided by SecureReview:

  • Sing Code Review Policy
  • Sing Code Reviews
  • Create a Verifable Chain of Reviews per Code Change
  • No server-side changes on web-based Git hosting services
  • Preserve the ease of use of web UI
  • Minimal impact on users experience
  • Efficient (add no significant delay)
